# Customer Tracker

A simple client tracker web app that demonstrates managing customer records using JavaScript arrays and objects.

## Overview

This project maintains a collection of customer records, including customer names, email addresses, and purchased items. It shows how to:

- create and store customer data
- add and remove records
- update existing customer information
- display summary information in the browser console

## Technologies Used

- **HTML**: Provides the basic structure and loads the JavaScript file
- **JavaScript**: Handles all the data manipulation and console output logic

## Files

- [index.html](index.html) — loads the application in a browser
- [cc_3.js](cc_3.js) — contains the customer data model and business logic

## How to Run the Project

1. Open [index.html](index.html) in a web browser.
2. Open the browser's developer console (F12 or right-click > Inspect > Console tab).
3. The script will run automatically and print customer summaries to the console.

## Script Behavior

The `cc_3.js` file:

- initializes an array of customer objects
- adds a new customer
- removes the first customer
- updates an existing customer email address
- adds a purchase entry for another customer
- logs customer details to the console

## What I Learned

This project helped me understand fundamental JavaScript concepts including:

- Working with arrays and objects
- Basic CRUD (Create, Read, Update, Delete) operations
- Array methods like `push()`, `shift()`, and `forEach()`
- Object property manipulation
- Console logging for debugging and output
- DOM-free browser scripting

## Notes

This repository is intended as a learning exercise to practice working with arrays, objects, and DOM-free browser scripting.
